# sturdy_potato
This is a cross over moving simple trading strategy for the apple stocks 
IT analyze the 20,50 day's of data and take the dicision's 
(SMA=Simple moving Average)
If the SMA'20>SMA'50 {BUY-short term trend is above long-term}
If the SMA'20<SMA'50 {SELL-short term trend is below long-term}
This is the classic "Golden Cross / Death Cross" strategy
